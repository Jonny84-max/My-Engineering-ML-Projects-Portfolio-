# JOU-Engineering-and-ML-Prototype-Projects-Portfolio-
Portfolio of my Prototype Engineering &amp; ML Practical Projects. It covers mini practical projects on predictive maintenance, hull biofouling prediction and optimization, and interactive student support chatbots. The projects highlight hands-on skills in Python, Streamlit, ML, Data Analysis and Engineering Systems.

# Features
a.  NNIT Student Support System: An interactive chatbot providing exam schedules, course registration, assignment and project deadlines as well as library information among othe students related matters.
b.  Predictive Maintenance for Marine Engines adaptable to Nigerian Navy Ship Engine Parameters: A machine learning system to forecast engine failures and optimize maintenance schedules.
c.   Hull Biofouling Predictor and Optimizer: Also Nigerian Navy Ships adapted model for predicting ship hull biofouling for improved ship performance and maintenance.
d.   User-Friendly Interactive Chatbot: A versatile interface demonstrating NLP and real-time responses in WAT.
